### 微信小程序各目录文件介绍

- app.json 全局属性配置，如微信小程序的菜单按钮
- app.js 小程序的全局js配置
- app.wxss 小程序的全局css

- utils文件夹是工具类js
- wxParse是用于解析html的js库，因为微信小程序不支持html，故需要它去做转换
- pages一共分index,directory,event,guides四大分类页和detail页组成
